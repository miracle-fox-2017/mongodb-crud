# mongodb-crud
crud with mongodb (native drive)

## REST API

List of user routes:

| Route               | HTTP          | Description      |
|:--------------------|:-------------:|:----------------|
| `/api/users`        |GET            | Get all the books|
| `/api/users`        |POST           | Create a book|
| `/api/users/:id`    |DELETE         | Delete a book|
| `/api/users/:id`    |PUT            | Update a book with new info|

## Usage
with only npm :
```
$ npm install
$ npm start
$ npm run dev
```
